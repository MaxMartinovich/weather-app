## Descrição
Aplicação utilizando a API da OpenWeather para buscar o clima de uma cidade
O aplicativo permite que o usuário adicione o nome de uma cidade e procurar no botão, além disso, pode ser utilizado um sistema de voz para procurar a cidade

## Tecnologias utilizadas
- HTML
- CSS
- JS

## Funcionalidades
- Procurar cidade por voz
- A API trás temperatura,temperatura  minima,umidade, sensação termica e um icone de temperatura
- Alterar o background para escuro ou claro
- Inteface simples

## Como executar o projeto
1ª - Instale um IDE, de preferência o VSCode porque roda muito bem com HTML,CSS e JS.

2ª - Copie o código disponibilizado

3ª - Execute a aplicação!
